# Smart Expense Tracker App

A simple yet powerful Smart Expense Tracker that helps users manage their daily expenses, track spending habits, and stay financially organized 📊.

---

🚀 Features :

📌 Add, edit, and delete expenses

📊 Track daily, weekly, and monthly spending

🏷️ Categorize expenses (Food, Travel, Bills, etc.)

📈 Visual insights with charts and reports

💾 Secure data storage

🌙 User-friendly and clean UI

---

🛠️ Tech Stack :

Frontend: HTML, CSS, JavaScript
